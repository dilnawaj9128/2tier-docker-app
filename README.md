# 2-Tier Docker Application

This project demonstrates a simple 2-tier web application using Docker.

## Tech Stack

- Node.js
- MySQL
- Docker
- Docker Compose

## Architecture

User → Node.js Container → MySQL Container

## Run the Project

Build and start containers:

docker-compose up -d

Check running containers:

docker ps

Open browser:

http://localhost:3000

## Features

- Node.js web server
- MySQL database container
- Docker networking
- Docker volumes for persistent storage
